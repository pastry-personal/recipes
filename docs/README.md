## 드레싱

### 이탈리안 드레싱

2022-02-13
정통 이탈리안 드레싱 레서피. 지금까지는 이것이 최고.

재료
올리브 오일 60ml
레몬즙 20ml
식초 10ml
마늘 튜브 작은 술 1/2
설탕 작은 술 1/2
소금 작은 술 1/2
블랙 페퍼 조금
드라이 허브(바질, 파슬리 로즈마리)

을 넣고 잘 흔들어 섞어준다.
